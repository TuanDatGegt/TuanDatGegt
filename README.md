End-to-end Retrieval-Augmented Generation system for medical question answering.

**Highlights**

- Built an end-to-end medical PDF processing pipeline.
- Designed a ChromaDB vector database with metadata management.
- Implemented Retrieval + Reranking + LLM generation pipeline.
- Developed a modular architecture supporting model swapping.
- Built an evaluation framework (EM, F1, Semantic Similarity, Context Recall).
- Developed an interactive Streamlit interface.

**Tech Stack**

`Python` `LangChain` `ChromaDB` `Sentence Transformers` `Hugging Face` `Streamlit`
